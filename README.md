# 🛒 Grocery Delivery Web Application

A scalable and real-time **Grocery Delivery Platform** built using modern full-stack technologies including **Next.js App Router, MongoDB, Redux Toolkit, Stripe, and Socket.IO**.

This platform enables customers to order groceries online, allows admins to manage products and orders efficiently, and helps delivery partners handle deliveries in real-time through dedicated dashboards.

---

# 🚀 Features

## 👤 Customer Features

- Secure Authentication & Authorization using **NextAuth**
- Browse and Search Grocery Products
- Add Products to Cart
- Smooth Checkout Experience
- Secure Online Payments with **Stripe**
- Real-Time Order Tracking using **Socket.IO**
- View Order History
- Responsive and User-Friendly Interface

---

## 🛠️ Admin Features

- Add, Edit, and Delete Grocery Products
- Manage Product Inventory & Categories
- View and Process Customer Orders
- Assign Orders to Delivery Partners
- Monitor Overall Platform Operations

---

## 🚚 Delivery Partner Features

- View Assigned Deliveries
- Accept or Reject Delivery Requests
- Update Delivery Status in Real-Time
- Delivery Workflow Management

---
